Forest is a production-grade [Filecoin](https://filecoin.io/) full node built by [ChainSafe Systems](https://chainsafe.io). Written in Rust, it delivers the same Filecoin protocol compatibility as the canonical Lotus node while using a fraction of the memory and CPU.

## What you can do with Forest

- **Sync the Filecoin blockchain** on mainnet, calibnet, or devnet
- **Host a JSON-RPC API** compatible with Lotus and standard Ethereum clients
- **Transfer FIL** and manage wallets with built-in key management
- **Generate and import snapshots** — up to 3× faster than Lotus with 10× less memory
- **Run as a bootstrap node** to strengthen the network
- **Monitor your node** with Prometheus metrics, health check endpoints, and Grafana dashboards

## Quick start

<Steps>
  <Step title="Install Forest">
    Pull the latest Docker image — no Rust toolchain required:

    ```bash
    docker pull ghcr.io/chainsafe/forest:latest
    ```
  </Step>
  <Step title="Start the daemon">
    Run Forest on calibnet with automatic snapshot download:

    ```bash
    docker run --init -it --rm \
      -v forest-data:/root/.local/share/forest \
      ghcr.io/chainsafe/forest:latest \
      --chain calibnet --auto-download-snapshot
    ```
  </Step>
  <Step title="Check sync status">
    In a second terminal, check that the node is syncing:

    ```bash
    docker exec -it <container> forest-cli sync status
    ```
  </Step>
</Steps>
